Ophthalmology Capacity Optimization in R

Overview

This project uses R to explore healthcare capacity allocation for ophthalmology services.

It compares different scheduling and optimization approaches to support better use of limited clinical capacity and reduce patient backlog.

Tools Used

* R
* lpSolve
* Simulated Annealing
* Linear Programming
* Data Visualization

What This Project Shows

* Healthcare operations analysis
* Capacity planning
* Scheduling optimization
* Cost comparison across methods
* Visual reporting in R

Example Outputs

The repository includes selected charts showing weekly capacity, cost breakdowns, and total weekly cost comparisons.

How to Run

Open and run:

ophthalmology_capacity_optimization.R
